# 酒店管理系统需求分析

- 功能：

1. 屏幕上出现一个界面，让操作员能够方便的选择所需要进行的操作，包括登记入住、查询房间的入住情况、查询当前费用、结账退房等操作。
2. 对不同标准的房间实施不同的收费标准，也可以按天收费或按小时收费，可根据顾客需要在登记的入住的时候进行选择。
3. 在结账退房时，根据入住的时间，计费方式和房间单价计算出总费用。

- 要求：用C++实现，先确认需求，使用面向对象的设计方法，对房间类别进行抽象化，基类room,派生出大床房、标间等，各自维护入住时间、型号、计费方式等属性，并实现相应的功能函数。设计一个管理类Hotel，负责管理房间的登记入住、查询和结账退房等操作。界面可以使用简单的命令行界面，提供用户友好的交互方式。

- 实现步骤：

1. 先逐一完成房间类别的设计和实现，然后实现Hotel类的功能，最后设计命令行界面进行交互。确保代码结构清晰，易于维护和扩展。
2. 在实现过程中，注意处理各种边界情况，例如房间已满、输入错误等，确保程序的健壮性和用户体验。
3. 使用qt绘制用户界面，先保证功能完善且正确，在按照苹果风格设计按键等元素，提升界面美观度和用户体验。
4. 最后进行测试，确保所有功能正常运行，并根据测试结果进行必要的调整和优化。
